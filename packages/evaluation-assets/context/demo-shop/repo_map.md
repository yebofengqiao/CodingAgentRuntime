# Demo Shop Repo Map

- `apps/demo-shop/src/pages/order-detail/` contains order detail UI and related tests.
- `apps/demo-shop/src/pages/checkout/` contains checkout flow components and tracking hooks.
- `apps/demo-shop/src/locales/` holds feature-level i18n bundles.
- `packages/order-shared/` contains non-UI shared business utilities that can be edited.
- `packages/ui/` is protected shared UI code and should not be modified in these evals.
